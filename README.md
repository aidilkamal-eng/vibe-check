# Vibe Check

> An AI that doesn't just document your codebase — it tests whether *you* understand it, then fills in exactly the gaps it finds.

## How it works

1. **Ingest** — Point it at a repo; it builds a model of structure, flows, and intent.
2. **Quiz** — It asks targeted questions about *your* code — not trivia, but the decisions and tradeoffs that matter.
3. **Diagnose** — Wrong or vague answers reveal specific gaps in understanding (yours or the codebase's).
4. **Fill in** — It generates documentation, diagrams, or explanations only for what's actually missing.
